Object (Drawable)

--------

  * Translate
  * Place
  * Update
  * Draw
  * Constructor
  * Destructor
  * Set scale
  * Get scale

Attributes:
-----------
  * Model
  * Texture/Color
  * Vertices
  * Reflectivity
  * Matrix
  * Position
